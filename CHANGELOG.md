## [1.0.0] - 2026-03-22

### Added（初回リリース — 全機能実装）

#### エリア検索機能（FR-01）
- テキスト入力によるエリア名サジェスト表示（部分一致フィルタ）
- キーボードナビゲーション対応（ArrowDown/Up、Enter、Escape）
- 入力クリアボタン（x ボタン）
- 対応エリアなし時のメッセージ表示
- エリアカードからの直接選択

#### スポット一覧表示機能（FR-02）
- スポットカード形式表示（スポット名・カテゴリ・簡易説明・所要時間）
- カテゴリアイコン表示（神社仏閣 / 自然 / グルメ / 体験 / ショッピング / 歴史 / 景観）
- スポットのルートへの追加/削除トグル（視覚的選択状態表示）
- aria-pressed によるアクセシビリティ対応

#### 地図表示機能（FR-03）
- Leaflet + OpenStreetMap による地図表示
- エリア中心座標への自動フォーカス
- 全スポットのマーカーピン表示
- マーカークリックによるポップアップ（スポット名 + 説明）
- 選択中スポットのマーカー色変更（青 → オレンジ）
- Next.js dynamic import (ssr: false) による SSR 回避

#### 周遊ルート表示機能（FR-04）
- 2 スポット以上選択時のポリライン描画（色: #3B82F6、太さ: 3px、opacity: 0.7）
- マーカーへの訪問順番号ラベル表示
- ルート情報パネル（選択スポット一覧 + スポット数）
- 上下ボタンによる訪問順並び替え
- 「ルートをクリア」ボタンによる全選択解除

#### レスポンシブ対応（FR-05）
- モバイル（〜767px）: スポット一覧 / 地図のタブ切り替え表示
- タブレット / デスクトップ（768px〜）: 左カラム（スポット一覧）+ 右カラム（地図）2 カラムレイアウト
- MobileTabSwitcher コンポーネントによる選択数バッジ表示

#### データ
- 5 エリア対応モックデータ整備
  - 東京（7 スポット）: 浅草寺、東京スカイツリー、新宿御苑 等
  - 京都（7 スポット）: 金閣寺、伏見稲荷大社、嵐山竹林 等
  - 大阪（7 スポット）: 大阪城、道頓堀、通天閣 等
  - 沖縄（7 スポット）: 首里城、美ら海水族館、古宇利島 等
  - 北海道（7 スポット）: 時計台、大通公園、富良野ラベンダー 等

#### 共通 UI
- ヘッダー（アプリタイトル + ホームリンク）
- フッター（著作権表示 + OpenStreetMap 帰属表示）
- BackButton（エリア検索画面への戻り）

#### ビルド・インフラ
- Next.js App Router + TypeScript strict mode
- 静的エクスポート設定（`output: 'export'`）
- Vercel デプロイ設定
- 5 エリア分の `generateStaticParams` による SSG

#### テスト
- Jest 29 + React Testing Library によるテスト整備
- テストスイート: 19 / テストケース: 155（全 Pass）
- ステートメントカバレッジ: 96.58%（目標 80% を大幅超過）
- Leaflet モック（`__mocks__/`）による地図コンポーネントのテスト対応
